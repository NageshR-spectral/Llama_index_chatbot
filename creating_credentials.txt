## Creating credentials for the DataStax database 

website - https://astra.datastax.com/org/

1. sign up (Free account)
2. Create free database
    a. database name
    b. provider (aws,gcp,azure)
    c. select region
3. after succefully creating database you can find the database details 
   at the Right side of the page.
4. copy API endpoint.
5. Click on setting icon of application token and download the json file 
   of token.
6. In region , click on three dots and download SCB file as zip file.
7. Move these two files in credential folder.
